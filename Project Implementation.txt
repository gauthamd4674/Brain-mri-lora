🧠 PHASE 3 — Understand Model Components (Day 5)

From Stable Diffusion:

3 key parts:
VAE → compress image → latent space

Text Encoder (CLIP) → encode prompt

🔧 PHASE 4 — LoRA Setup (Day 6–7)
💡 What LoRA does:

Instead of training full model:

Adds small trainable matrices to attention layers
⚙️ Inject LoRA into UNet

Using peft:

from peft import LoraConfig

config = LoraConfig(
    r=8,
    lora_alpha=16,
    target_modules=["to_q", "to_v"],  # attention layers
    lora_dropout=0.1
)

Attach to UNet.

🔁 PHASE 5 — Training Pipeline (CORE PART) (Day 8–12)

This is the heart of the project.

🔄 Training Loop Logic
Step-by-step:
1. Load image
image = load_image()
2. Encode to latent
latents = vae.encode(image).latent_dist.sample()
latents *= 0.18215
3. Sample noise
noise = torch.randn_like(latents)
4. Sample timestep
t = torch.randint(0, num_timesteps, (batch_size,))
5. Add noise
noisy_latents = scheduler.add_noise(latents, noise, t)
6. Get text embeddings
text_embeddings = text_encoder(prompt)
7. Predict noise
noise_pred = unet(noisy_latents, t, encoder_hidden_states=text_embeddings)
8. Compute loss
loss = F.mse_loss(noise_pred, noise)
9. Backprop
loss.backward()
optimizer.step()
⚠️ Key Insight

The model learns:

“Given a noisy latent → predict the noise → remove it”
